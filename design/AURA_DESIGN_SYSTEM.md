# Aura design tokens (summary)

Canonical tokens live in `web/src/theme.css` (CSS variables). This file tracks intent for collaborators.

- Canvas / lavender wash backgrounds for calm default state
- Blush accent for primary actions and SOS idle state
- Status colors: ok / warn / alert
- Large card radius (~20px), soft shadow, glass cards via semi-transparent white

Refer to screen specs for route-level layout expectations.
